# Dialogue Editor - 2人掛け合いシナリオエディタ

2人の掛け合いに特化した機能限定版のシナリオエディタです。

## 主な機能

### ✨ キャラクター登録機能
- 最大4人までキャラクターを登録可能
- 登録した名前ボタンをクリックすると「キャラ名：　」が入力される
- キャラクターボタンの×マークで削除可能

### 🔢 連番機能
- 「キャラ名：」で始まる行の頭に半角数字3桁の番号を一括で自動付与
- 既存の番号は自動的に更新される

### 📊 文字数表示機能
- 総文字数をリアルタイム表示
- キャラクターごとの文字数を表示（：やキャラ名、括弧を除く）
- 行数も表示

### 🎬 システム入力ボタン
- **ト書きボタン**: `// ` + 半角空白を入力
- **シーンボタン**: `〇 ` + 半角空白を入力
- **暗転ボタン**: `【暗転】` + 改行を入力

### 🏠 プロジェクト管理
- **ウェルカムウィンドウ**: 起動時にプロジェクト選択画面を表示
- **プロジェクト保存**: ローカルストレージに自動保存（手動保存も可能）
- **プロジェクト読み込み**: 過去のプロジェクトを開く
- **TXT書き出し**: シナリオをテキストファイルとして出力
- **🏠ボタン**: ウェルカムウィンドウに戻る

### 📝 プロジェクト名入力欄
- 入力した内容がプロジェクト保存時の名前に反映
- TXT書き出し時のファイル名にも使用される

## デバイス対応

### 📱 スマホ・タブレット対応
- レスポンシブデザインで画面サイズに自動調整
- タッチ操作に最適化されたボタンサイズ
- PWA対応でホーム画面に追加可能

### 💻 デスクトップ対応
- 広い画面では2カラムレイアウトで効率的に作業
- キーボードショートカットに対応

## PWA（アプリ化）機能

### スマホ・タブレットでアプリとして追加
1. Safariで開く（iOS）またはChromeで開く（Android）
2. 共有ボタンから「ホーム画面に追加」を選択
3. アプリのようにホーム画面から起動可能

## 使い方

### 新規プロジェクト作成
1. ウェルカム画面で「新しいプロジェクトを作成」をクリック
2. プロジェクト名を入力
3. エディタ画面が開きます

### キャラクター登録
1. 右側のキャラクター登録欄に名前を入力
2. 「追加」ボタンをクリック
3. 登録されたキャラクターボタンが表示されます

### シナリオ入力
1. キャラクターボタンをクリックして「キャラ名：　」を入力
2. セリフを入力
3. システムボタンでト書きやシーンを追加

### 連番付与
1. ツールバーの「🔢 連番付与」ボタンをクリック
2. 「キャラ名：」で始まる行に自動的に番号が付与されます

### プロジェクト保存
- 自動保存機能により2秒ごとに自動保存
- ヘッダーの「💾 保存」ボタンで手動保存も可能

### TXT出力
- ヘッダーの「📄 TXT」ボタンでテキストファイルとして出力
- ファイル名はプロジェクト名が使用されます

## GitHub Pages での公開方法

1. GitHubリポジトリを作成
2. このフォルダの内容をリポジトリにプッシュ
3. Settings > Pages で「main」ブランチを選択
4. 公開されたURLにアクセス
